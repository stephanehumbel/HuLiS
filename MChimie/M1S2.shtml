On trouvera de nombreuses données dans le NIST : https://webbook.nist.gov/chemistry/

Recommendations : Créez votre répertoire personnel au bon endroit (dans M1S1/202x). Exemple cd M1S1/2023; mkdir MES_INITIALES, et allez-y (cd MES_INITIALES).

Quelques commandes Linux (ls, pwd) nécessaires

  • mkdir repertoire (make directory) crée un sous-répertoire dans le répertoire courant
  • cd repertoire (change directory) permet d'aller dans le répertoire-fils
  • cd .. permet d'aller dans le répertoire-père
  • pwd (print working directory) affiche la localisation complète du répertoire courant
  • ls (listing) affiche le nom des fichiers et sous-répertoire du répertoire courant
  • more fichier affiche le contenu du fichier sur la console
  • rm fichier efface le fichier
  • mv fichier1 fichier2 (move) change le nom de fichier1 en fichier2
  • cp fichier1 fichier2 (copy) copie fichier1 sous le nom fichier2
  • nedit fichier édite le contenu du fichier dans une fenetre, permet de le modifier
  • grep chaine file renvoie les lignes du fichier "file" dans lesquelles "chaine" apparait
  • which commande renvoie la localisation de la commande
  • man commande le manuel de la commande
Quelques logiciels et commandes maisons permettent de se simplifier la vie
  • agui visualiseur/builder (commercial)
  • gnuplot traceur de courbes (exemple pour un fichier 2 colonnes (2cols.data) gnuplot, puis dans gnuplot> taper plot '2cols.data' w lp lw 5 (lp pour linespoints lw pour linewidth)
  • Commandes maison accessibles par la commande ''. outils'' (ne pas oublier le point ''.'')
    • qq montre nos jobs et leur repertoire (squeue -U $USER,$1 --format %.7i%.30j%.3t%.8C%.55Z)
    • MXYZ fait un raccourci de l'optimisation (maison, gaussian)
    • xopt fait un raccourci de l'optimisation (maison, gaussian)
    • somm fait un sommaire des resultats d'un calcul (maison, gaussian)
    • ellipsoid calcule les 3 axes de l'ellipsoid qui inscrit la derniere geométrie d'un fichier .log (gaussian)
